This Project started as a python program to analyze a simple CSV file, As I developed the program I wanted to make an automated script that would ask the user for different information within the dataset that would later convert into a detailed analysis of the information within the CSV file. This automation could also be performed by extracting the information from a Database making way more simple the analysis of complex data into a series of queries/questions within the script for an easy and more complex analysis to that everyone can have access.
